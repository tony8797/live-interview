# live-interview
以 react + redux + redux-persist 建置抽獎程式

## How to use

Execute [`live-interview`](https://github.com/tony8797/live-interview) with [npm](https://docs.npmjs.com/cli/init) or [Yarn](https://yarnpkg.com/lang/en/docs/cli/create/):

```bash
npm install
# or
yarn 
```
